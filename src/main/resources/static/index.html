Observacoes:
  • A IDE utilizada para desenvolver o projeto foi o IntelliJ
  • Frontend feito em angular, caso nao consiga rodar o frontend na sua maquina, voce deve instalar o node v16.14 ou superior + o angular na versao 16
  • Verifique a instalacao com os comandos: node -v // npm -v // ng version
  • Para conseguir acessar o frontend do projeto, no terminal da sua IDE entre na pasta src/main/resources/templates/ado-pi e RODE O COMANDO: npm run start
  • Apos concluir, digite no browser: localhost:4200 que ele deve carregar a tela incial
  • para rodar o backend, apenas rode o projeto na IDE, ele obrigatoriamente tem que estar na porta 8080
